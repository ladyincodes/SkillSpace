# SkillSpace

SkillSpace is a platform where local businesses or individuals can list workshops (e.g., cooking classes, art sessions, or coding boot camps). Users can join workshops, and follow specific organizers to get updates on future events.
Developed with React and ASP.NET Core(C#).

## Tech-stack

C# (ASP.NET Core), SQL Server(Entity Framework).
React, Semantic-UI, TypeScript, Axios.
